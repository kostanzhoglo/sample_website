Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
blockquote blockquote include an indented blockquote paragraph
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header creates header section
div division create divisions in web page
ol ordered list reate ordered list
ul unordered list create unordered list

Inline Elements

Tag Name Purpose Example Result
br break inserts a line break There's a break at the end of this sentence.<br>This next sentence is on the following line. There's a break at the end of this sentence.
This next sentence is on the following line.
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion<strong> at least a billion
a anchor make hyperlink <a href="http://learnenough.com">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code make code text <code>a href<code> a href
span span used for styling. marks text. <span>Call me Ishmael.</span> Call me Ishmael.
li list item represents one item in ordered/unordered list <li>Vengeful whale</li> 1. Vengeful whale

Document Tags

Tag Name Purpose
html HTML Specifier Tells the browser that we are using HTML
head Document Header Specifies the title and other info about the page
body Document body Begins the body of the page
meta META Tag Specifies information about the document, like character set
title Title in Tab Puts a title in the tab of the page. Useful for SEO.
